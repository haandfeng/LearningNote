# 倍增法
Binary Lifting

---

倍增法（Binary Lifting），顾名思义就是翻倍。它能够使线性的处理转化为[[对数]]级的处理，大大地优化时间复杂度。

这个方法在很多算法中均有应用，其中最常用的是 [[RMQ问题]]和求 [[二叉树的最近公共祖先问题|LCA]]（最近公共祖先）。

##### [[RMQ问题]]
RMQ 是 Range Maximum/Minimum Query 的缩写，表示区间最大（最小）值。使用倍增思想解决 RMQ 问题的方法是 [[ST表]]。
