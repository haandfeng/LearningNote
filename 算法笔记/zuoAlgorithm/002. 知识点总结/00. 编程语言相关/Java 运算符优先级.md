# Java 运算符优先级

---

运算符的优先级（由高到低）

```text
1	括号	()、[ ]、.
2	非、正，负号	!、+(正)、-(负)
3	自增，自减	++、--
4	乘，除，取余	*、/、%
5	加，减	+、-
6	移位运算	<<、>>、>>>
7	大小关系	<、<=、>、>=
8	相等关系	==、!=
9	按位与	&
10	按位异或（在java中，这不是幂次方的符号）	^
11	按位或	|
12	逻辑与(短路与)	&&
13	逻辑或(短路或)	||
14	条件运算(三目运算)	? :
15	赋值运算	=、+=、-=、*=、/=、%=
16	位赋值运算	&=、|=、^=、~=、<<=、>>=、>>>=
```


```java
public static void main(String[] args) {  
    int n = 5;  
    if( 3 > n>>1) { // NOTE: 右移优先级比 > 高  
        System.out.println("OK");  
    } else {  
        System.out.println("NG");  
    }  
      
    int L = 0;  
    int R = n;  
    int mid = L + ((R-L)>>1); // NOTE: + 优先级比 >> 高, 所以要括号  
}

```
