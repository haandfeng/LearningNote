# 求n个矩形的总面积

#扫描线 
#线段树 

---

https://www.luogu.com.cn/problem/P5490

求n个矩形的面积。

输入格式
第一行一个正整数n。

接下来 nn 行每行四个非负整数 x_1, y_1, x_2, y_2 ，表示一个矩形的左下角坐标为 (x_1, y_1)，右上角坐标为 (x_2, y_2)。

输出格式
一行一个正整数，表示n个矩形的并集覆盖的总面积。

```
输入输出样例
输入1
2
100 100 200 200
150 150 250 255
输出1
18000

说明/提示
对于20%的数据，1≤n≤1000。
对于100%的数据1≤n≤10^5, x_1<x_2≤10^9, y_1<y_2≤10^9
```
